NumberLink

Автор: Кокорина Елизавета

Описание
Программа, которая решает задачи для игры NumberLink
Данная программа представляет собой игру в формате клавиатурного тренажера. Программа рассчитана на одного и более позьзователей, с выбором победителя, если такие есть.
В игре необходимо вводить имя монстрика, которого нужно убить. 

Требования
Python версии не ниже 3.7

Состав
Консольная версия решателя: numberlink.py
Консольная версия генератора: numberlink_generator.py
Тесты:  numberlink_test.py
	map_test.py
	generator_test.py
	drawing_test.py

Справка по командам: help [команда]

На данные модули (numberlink, Graph, Map, numberlink_generator, Video) написаны тесты. Покрытие по строкам составляет около 89%
