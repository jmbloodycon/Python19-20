Клавиатурный тренажер

Автор: Кокорина Елизавета

Описание
Данная программа представляет собой игру в формате клавиатурного тренажера. Программа рассчитана на одного и более позьзователей, с выбором победителя, если такие есть.
В игре необходимо вводить имя монстрика, которого нужно убить. 

Требования
Python версии не ниже 3.6

Состав
Глафическая версия: game_window.py
Тесты:  typing_tutor_test.py
	logic_test.py

Пример запуска: ./game_window.py

Справка по командам: help [команда]

Функция get_object_blit(text, x, y, size, font, color) создает объект для отрисовки текста по заданным параметрам 
Функции класса TypingTutor
Функция run() класса TypingTutor запускает все составные части программы.
Функция request_players_info(digit) запрашивает у пользователя его имя или количество игроков в графическом режиме.
Функция make_winner() отрисовывает имя победителя.
Функция choose_winner(players) выбирает победителя из заданного листа игроков по времени, если таковые есть

Функции класса Game
Функция new_game() отрисовывает монстриков и следит за расположением их на экране, а так же отслеживает собития клавиатуры. Основная функция игры для пользователя.
Функция kill_monster(name, monsters) убивает монстрика из набора monsters в соответствии с введенным именем

Функции класса Monster
Функция def kill_monster() зануляет скорость монстрика и убивает его.

На данные модули (Monster, Player, game_window) написаны тесты. Покрытие по строкам составляет около 27% из низ покрытие логики около 98%
