KV-Storage

Автор: Кокорина Елизавета

Описание
	Данная программа представляет собой локальное храниличще данных в формате "ключ-значение". Доступны операции: добавление, удаление по ключу, получение значения по ключу.

Требования
	Python версии не ниже 3.7

Состав
	Тесты:  KV_storage_test.py

Примеры команд: 
	add lol kek - добавление значения 'kek' по ключу 'lol'
	del lol - удаление значения по ключу 'lol' и самого ключа
	get lol - получение значения по ключу 'lol'
	exit - выход из программы
