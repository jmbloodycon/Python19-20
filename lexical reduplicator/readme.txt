Лексический редупликатор

Автор: Кокорина Елизавета

Описание
Данная программа анализирует вводимые пользователем 3 префикса и изменяет предложения, записывая новые слова с подставленным префиксом через дефис. 

Требования
Python версии не ниже 3.6

Состав
Консольная версия: lexicalReduplicate.py
Тесты: lexicalReduplicate_test.py

Пример запуска: ./lexicalReduplicate.py
Пример работы: Пример: предложение "Привет, меня зовут Евгений" с тремя введенными префиксами "ола", "хе" и "пелу" программа преобразует в предложение "Привет-хивет, меня-оляня зовут-хёвут Евгений-пелугений"

Справка по командам: help [команда]

Функция run() класса LexicalReduplicate запускает все составные части программы.
Функция request_prefixes() запрашивает у пользователя три префикса.
Функция make_reduplication() делает редуплитацию текста из файла "re.txt" в соответствии с выбранным для каждого слова префиксом и возвращает результат в виде строки.
Функция word_analyze(word, prefix) анализирует преобразование слова в соответствии с заданным префиксом и проверяет на соответствие лексикоорфоэпической составляющей. Возвращет измененное слово с дефисным написанием.
Функция change_prefix_form(prefix, word) изменяет форму префикса в соответствии со словом, которое требуется заменить. Возвращет форму префикса.
